# hackernoon2.0


## Requirements

```bash
Node >= 8
```

### Install
```bash
git clone git@github.com:DavidAbram/hackernoon2.0.git
```

```bash
mkdir loom && cd loom
curl https://raw.githubusercontent.com/loomnetwork/loom-sdk-documentation/master/scripts/get_loom.sh | sh

./loom init
./loom run
```

```bash
# Open second terminal
npm run deploy
npm test

npm serve
```
